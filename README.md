# 👥 CRUD de Usuários

Aplicação web simples que permite cadastrar, listar, editar e excluir usuários.

## 🚀 Funcionalidades
- Cadastro de usuários
- Edição de dados
- Exclusão com confirmação
- Armazenamento no navegador (localStorage)

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## 🌐 Acesse
https://seuusuario.github.io/crud-usuarios/

## 📌 Objetivo
Projeto desenvolvido para praticar lógica de programação, manipulação de DOM e persistência de dados.
